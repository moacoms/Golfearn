# X 스레드: 연습장 처음 가는 법

## 1️⃣ (훅)
```
42살에 연습장 처음 갔을 때요?

문 앞에서 10분 서성였어요.

"뭘 어떻게 해야 하지?"
"다들 나 쳐다보겠지?"

같은 고민 하시는 분들을 위해 정리했어요.

🧵
```

## 2️⃣
```
연습장 종류부터 알려드릴게요.

스크린: 실내, 혼자 방에서 침 (덜 창피)
실외: 진짜 공 날아감, 옆 사람 다 보임

처음이면 스크린 강추.
저는 3개월 스크린만 다녔어요.
```

## 3️⃣
```
"뭘 들고 가야 해요?"

필수: 편한 운동화, 편한 옷, 골프 장갑
선택: 클럽 (없으면 빌려요)

골프화 필요 없어요.
청바지도 OK.
장갑만 본인 거 하나 사세요.
```

## 4️⃣
```
"공은 어떻게 나와요?"

스크린: 자동으로 나와요
실외: 카드 찍으면 나와요

저는 첫날 공 나오는 법 몰라서
3분 동안 멍하니 서 있었어요 ㅋㅋ

모르면 직원한테 물어보세요.
다들 친절해요.
```

## 5️⃣
```
"첫날 뭐부터 쳐야 해요?"

1. 스트레칭 5분
2. 7번 아이언 30분 ← 이게 기본
3. 드라이버 15분
4. 짧은 클럽 10분

드라이버 먼저 치고 싶겠지만
7번 아이언부터 시작하세요.
```

## 6️⃣
```
첫날 창피했던 순간들

• 10구 중 1구만 맞음
• 헛스윙 연발
• 공이 옆으로 갔음
• 옆 사람이 엄청 잘 치는데 나만...

다 정상이에요.
저도 첫날 100구 치고 5구만 맞았어요.
```

## 7️⃣
```
비용 정리

스크린: 30분 5,000~8,000원
실외: 100구 5,000~10,000원

장갑: 1~2만원 (중고면 5천원)
렌탈 클럽: 보통 무료 or 2~3천원

첫달 총 비용?
주 2회 기준 월 5~10만원 정도.
```

## 8️⃣
```
연습장 에티켓

✅ 옆 사람 치는 동안 기다리기
✅ 조용히 하기
✅ 뒷정리 하고 나가기

❌ 옆 사람한테 먼저 조언하기

저도 모르는 아저씨한테
20분 동안 레슨 받은 적 있어요...
요청 안 한 조언은 피하세요.
```

## 9️⃣
```
정리하면

1. 처음엔 스크린 추천

다 그랬어요.
저도 그랬고요.
```
