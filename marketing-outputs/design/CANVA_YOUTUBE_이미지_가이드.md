# Canva로 YouTube 이미지 만들기 - Golfearn

> Canva 무료 버전으로 충분히 제작 가능합니다.

---

## YouTube에 필요한 이미지 목록

| 이미지 | 사이즈 | 용도 |
|--------|--------|------|
| 프로필 사진 | 800 x 800 px | 채널 아이콘 |
| 배너 이미지 | 2560 x 1440 px | 채널 상단 배너 |
| 썸네일 | 1280 x 720 px | 영상 썸네일 |

## 1. 프로필 사진 만들기

### Canva 설정
```
1. canva.com 접속 (무료 가입)
2. [디자인 만들기] → [사용자 지정 크기]
3. 800 x 800 px 입력
4. [새 디자인 만들기]
```

### 디자인 가이드
```
┌─────────────────────┐
│                     │
│     ┌───────┐       │
│     │  G    │       │
│     │ 골프런 │       │
│     └───────┘       │
│                     │
└─────────────────────┘
```

### 제작 순서
```
1. 배경색 설정: #10B981 (녹색)
2. [텍스트] → "G" 추가
   - 폰트: Montserrat Bold
   - 크기: 300pt
   - 색상: 흰색
3. 또는 [요소] → 골프 아이콘 검색
4. 다운로드 (PNG)
```

### 대안 1: 텍스트 로고
```
배경: #10B981 (녹색)
텍스트: "G" 또는 "골프런"
폰트: 굵은 고딕
```

### 대안 2: 아이콘 + 텍스트
```
골프공 아이콘 + "G" 조합
```

## 2. 배너 이미지 만들기

### Canva 설정
```
1. [디자인 만들기] → "YouTube 채널 아트" 검색
2. 또는 사용자 지정: 2560 x 1440 px
```

### 디자인 가이드 (안전 영역 중요!)
```
┌──────────────────────────────────────────────────────────┐
│                    (TV에서만 보임)                        │
│  ┌────────────────────────────────────────────────────┐  │
│  │              (데스크톱에서 보임)                    │  │
│  │  ┌──────────────────────────────────────────────┐  │  │
│  │  │                                              │  │  │
│  │  │     Golfearn 골프런                          │  │  │
│  │  │                                              │  │  │
│  │  │     늦게 시작해도 괜찮아, 함께라면           │  │  │
│  │  │                                              │  │  │
│  │  │     🏌️ 골린이 전용 플랫폼                    │  │  │
│  │  │                                              │  │  │
│  │  └──────────────────────────────────────────────┘  │  │
│  │                (모바일 안전 영역)                  │  │
│  └────────────────────────────────────────────────────┘  │
└──────────────────────────────────────────────────────────┘

핵심 콘텐츠는 중앙 1546 x 423 px 안에 배치!
```

### 제작 순서
```
1. 배경 설정
   옵션 A: 단색 녹색 (#10B981)
   옵션 B: 골프장 사진 + 어두운 오버레이
   옵션 C: 그라데이션 (검정 → 녹색)

2. 메인 텍스트 (중앙)
   "Golfearn 골프런"
   - 폰트: Montserrat Bold + Noto Sans KR
   - 크기: 150pt
   - 색상: 흰색

3. 서브 텍스트
   "늦게 시작해도 괜찮아, 함께라면"
   - 크기: 60pt
   - 색상: 흰색 또는 연녹색

4. 아이콘/장식 (선택)
   - 골프공, 클럽 아이콘
   - 체크마크 아이콘
```

### 추천 Canva 템플릿
```
검색어: "YouTube banner minimal"
검색어: "YouTube banner green"
검색어: "YouTube banner sports"
→ 마음에 드는 템플릿 선택 후 수정
```

## 3. 썸네일 만들기

### Canva 설정
```
1. [디자인 만들기] → "YouTube 썸네일" 검색
2. 1280 x 720 px
```

### 쇼츠 1 썸네일 예시: "골린이의 고민"
```
┌─────────────────────────────────────┐
│                                     │
│   골프 시작하고                     │
│   싶은데...        [골프공 이미지]  │
│                                     │
│   😰 뭐부터 해야해?                 │
│                                     │
└─────────────────────────────────────┘
```

### 제작 순서
```
1. 배경
   - 색상: 검정 또는 진한 녹색
   - 또는 골프 관련 스톡 이미지 + 어두운 오버레이

2. 메인 텍스트
   "골프 시작하고 싶은데..."
   - 폰트: 굵은 고딕
   - 크기: 화면의 40% 차지
   - 색상: 흰색 또는 노란색
   - 테두리/그림자 추가 (가독성)

3. 강조 요소
   - 이모지: 😰 또는 🤔
   - 화살표, 느낌표 등

4. 로고 (선택)
   - 우측 하단에 작게 "골프런" 로고
```

### 썸네일 공식 (클릭률 높이는)
```
[충격적 숫자] + [질문] + [이모지]

예시:
"300만원?" + "진짜 이만큼 들어?" + 😱
"42살에 시작" + "늦은 거 아닌가요?" + 🤔
"100타 이상만" + "초보끼리 모여라" + 🏌️
```

검정:         #111827
흰색:         #FFFFFF
연한 회색:    #F9FAFB

### Canva에서 색상 입력 방법
```
1. 색상 선택 클릭
2. [+] 버튼 또는 색상 코드 입력란

```

## 7. Canva 무료 vs Pro

### 무료로 충분한 것
```
✅ 프로필 사진
✅ 배너 이미지
✅ 썸네일
✅ 기본 템플릿
✅ 기본 요소/아이콘
```

### Pro가 필요한 것 (선택)
```
⭐ 브랜드 키트 (색상/폰트 저장)
⭐ 배경 제거
⭐ 프리미엄 템플릿
⭐ 더 많은 스톡 이미지
```
