## STEP 2: 장면별 제작 (25분)

### 🎬 장면 1: 인트로 (0초~3초)

**배경 설정:**
```
1. [+] 버튼 → [스톡] → 검은색 배경 추가
2. 길이: 3초로 조정
```

**텍스트 추가:**
```
[텍스트] 버튼 터치 → 아래 내용 복사/붙여넣기:

골프 시작하고 싶은데...
```

**텍스트 설정:**
```
- 폰트: Noto Sans KR Bold (또는 기본 굵은 고딕)
- 크기: 80pt (화면 70% 차지하도록)
- 색상: #FFFFFF (흰색)
- 정렬: 중앙
- 애니메이션: [인] 페이드인, [아웃] 페이드아웃
```

---

### 🎬 장면 2: 고민 나열 (3초~12초)

**4개 텍스트를 순서대로 등장시킵니다:**

| 시간 | 텍스트 | 지속시간 |
|------|--------|----------|
| 3초~5초 | 클럽 뭐 사야 해? | 2초 |
| 5초~7초 | 레슨 어디서 받아? | 2초 |
| 7초~9초 | 혼자 필드 어떻게 가? | 2초 |
| 9초~12초 | 다 너무 비싸... | 3초 |

**각 텍스트 설정:**
```
- 폰트: Noto Sans KR Bold
- 크기: 70pt
- 색상: #FFFFFF
- 애니메이션: [인] 타이핑 효과
- 배경: 검은색 유지
```

**팁:** 각 텍스트를 별도 레이어로 추가하고, 타임라인에서 시작점 조정

## STEP 3: 배경음악 추가 (3분)

### 추천 BGM (CapCut 내장)

```
1. 하단 [오디오] 터치
2. [음악] → [분위기] → "Inspiring" 또는 "Happy" 선택
3. 추천 트랙:
   - "New Beginning" (잔잔한 시작)
   - "Good Vibes" (밝은 느낌)
   - "Motivation" (동기부여)
4. 볼륨: 30-40%로 조정 (목소리 있으면 더 낮게)
```

### 효과음 추가 (선택)

```
- 텍스트 등장 시: "Whoosh" 효과음
- 체크마크 등장 시: "Pop" 효과음
- 위치: [오디오] → [효과음]
```

## 화면녹화 가이드 (미리 준비)

### golfearn.com 화면녹화 방법

**PC에서:**
```
1. Chrome 브라우저로 golfearn.com 접속
2. F12 → 모바일 뷰 (iPhone 12 Pro 선택)
3. Windows: Win + G (게임바) 또는 OBS
4. Mac: Cmd + Shift + 5
5. 녹화 시작 후 천천히 스크롤 (7초)
```

**녹화할 화면:**
```
1. 메인페이지 (히어로 섹션) - 2초
2. 조인 매칭 페이지로 이동 - 2초
3. 중고거래 페이지로 이동 - 2초
4. 레슨프로 페이지 - 1초
```
