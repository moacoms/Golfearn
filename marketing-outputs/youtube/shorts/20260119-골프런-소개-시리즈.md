## 쇼츠 1: 골린이의 고민 (공감형)

### 컨셉
골린이가 겪는 현실적인 고민을 나열 → Golfearn 소개

### 대본 (60초)
```
[화면: 검은 배경 + 흰 텍스트, 한 줄씩 등장]

"골프 시작하고 싶은데..."

[텍스트 전환]
"클럽 뭐 사야 해?"
"레슨 어디서 받아?"
"혼자 필드 어떻게 가?"
"다 너무 비싸..."

[텍스트 전환, 강조]
"저도 42살에 시작할 때
똑같았어요"

[화면: Golfearn 웹사이트 화면녹화]
"그래서 만들었습니다"

[로고 등장]
"Golfearn - 골프런"

[텍스트]
"골린이 전용 플랫폼"

✅ 100타 이상끼리 조인
✅ 중고 클럽 거래
✅ 동네 레슨프로 찾기

[마지막 화면]
"늦게 시작해도 괜찮아, 함께라면"

👇 프로필 링크
```

### 편집 팁
- 텍스트 등장 속도: 1-2초마다 전환
- 배경음악: 잔잔한 피아노 or 경쾌한 BGM
- 폰트: 굵고 읽기 쉬운 고딕체
- 화면녹화: Golfearn 메인페이지 스크롤

## 쇼츠 2: 골프 시작 비용 현실 (정보형)

### 컨셉
골프 시작 비용에 대한 현실적인 정보 제공

### 대본 (60초)
```
[화면: 충격적인 숫자]
"골프 시작하는데 300만원?"

[텍스트 전환]
"아니요, 현실은..."

[항목별 텍스트 + 금액]
📦 중고 풀세트: 30-50만원
👨‍🏫 레슨 3개월: 30-40만원
🏌️ 연습장 (주2회): 월 5-8만원
⛳ 첫 라운딩: 15-20만원

[계산]
"첫 3개월 총 비용"
"약 80-120만원"

[강조]
"월 30만원이면 시작 가능"

[전환: Golfearn 화면]
"더 아끼고 싶다면?"

[Golfearn 기능 보여주기]
• 중고 클럽 거래 → 장비비 50% 절약
• 골린이 조인 → 그린피 나눠내기
• 레슨프로 비교 → 합리적인 레슨

[마지막]
"골프런에서 똑똑하게 시작하세요"
👇 링크
```

### 편집 팁
- 숫자는 크게, 애니메이션 효과
- 계산기 효과음 활용
- Golfearn 화면은 실제 사이트 녹화

5️⃣ 모르는 아저씨가 레슨해줌

[텍스트 전환]
"다 겪어봤어요 😅"

[해결책]
"그래서 준비했습니다"

[Golfearn 화면]
"골린이끼리만 모이는 곳"

• 100타 이상만 조인
• 초보끼리 편하게
• 창피할 일 없음

[마지막]

### 편집 팁
- 각 항목에 이모지 효과

## 제작 가이드

### 필요한 것
1. **화면 녹화**: Golfearn 웹사이트 (www.golfearn.com)
2. **편집 앱**: CapCut (무료)
3. **배경음악**: CapCut 내장 음악 or 유튜브 오디오 라이브러리
4. **목소리**: 직접 녹음 or AI 음성 (선택)

### CapCut 편집 순서
```
1. 새 프로젝트 → 9:16 비율
2. 검은 배경 추가
3. 텍스트 입력 (크게, 가운데)
4. 애니메이션 효과 추가
5. 화면 녹화 영상 삽입
6. 배경음악 추가
7. 내보내기 (1080p)
```

### 텍스트 스타일
- 폰트: 고딕 (굵게)
- 크기: 화면의 70% 차지
- 색상: 흰색 (검은 배경) 또는 검정 (밝은 배경)
- 그림자: 약간 추가

### 업로드 설정
```
제목: [질문형 or 숫자] + 핵심 키워드
예: "골프 시작 비용 현실 💰 #골린이"

설명:
골린이를 위한 플랫폼, 골프런
👉 www.golfearn.com

#골린이 #골프초보 #골프시작 #Shorts

태그: 골린이, 골프초보, 골프시작, 골프입문, 골프비용
```
