# YouTube Shorts: 연습장 처음 가는 법 (60초)

- **제목**: 연습장 처음 가는 골린이 필수 시청 #골린이 #골프초보
- **길이**: 60초
- **형식**: 세로 9:16

---

## 대본

### 0-5초 (훅)
```
[정면 보며, 긴장한 표정]
"연습장 문 앞에서 10분 서성인 적 있으시죠?"
[자막: 🚪 연습장 앞에서 10분...]
```

### 5-15초 (공감)
```
[손 흔들며]
저도요.
42살에 처음 갔는데
뭘 어떻게 해야 할지 모르겠더라고요.
[자막: 뭘 어떻게...?]
```

### 15-25초 (핵심 1)
```
[손가락 1 펴며]
첫 번째, 스크린 연습장 가세요.
[B-roll: 스크린 연습장]
혼자 방에서 치니까 덜 창피해요.
[자막: ✅ 스크린 연습장 추천]
```

### 25-35초 (핵심 2)
```
[손가락 2 펴며]
두 번째, 장갑만 들고 가세요.
[장갑 보여주며]
클럽은 빌려줘요.
[자막: ✅ 장갑만 챙기기]
```

### 35-45초 (핵심 3)
```
[손가락 3 펴며]
세 번째, 7번 아이언부터 치세요.
[클럽 보여주며]
드라이버 먼저 치고 싶겠지만 참으세요.
[자막: ✅ 7번 아이언부터]
```

### 45-55초 (위로)
```
[부드러운 표정]
첫날 10구 중 1구만 맞아도 성공이에요.
[자막: 10구 중 1구 = 성공 🎉]
저도 첫날 100구 치고 5구만 맞았어요.
[자막: 나도 5구...]
```

### 55-60초 (CTA)
```
[웃으며]
창피해도 괜찮아요, 다 그랬어요.
[자막: 늦게 시작해도 괜찮아, 함께라면]
프로필 링크에서 같이 칠 골린이 찾아보세요!
[자막: 👇 프로필 링크]
```
