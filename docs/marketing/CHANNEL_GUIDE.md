## 채널별 운영 전략

### YouTube

#### 컨셉
- **메인 영상** (8-15분): 심층 가이드
- **쇼츠** (60초): 핵심 팁

#### 초기 영상 10개 (우선순위)
1. 채널 소개 "42살에 골프 시작한 이유"
2. 연습장 처음 가는 법
3. 첫 클럽 세트 고르는 법
4. 레슨프로 고르는 기준
5. 골프 시작 비용 총정리
6. 중고 클럽 구매 가이드
7. 혼자 조인 가는 법
8. 골린이 필드 에티켓
9. 스코어 100타 깨는 법
10. 골린이 1년 후기

#### 영상 제작 팁
- **촬영**: 스마트폰 OK (가로 촬영)
- **편집**: CapCut (무료) / 프리미어 러시
- **썸네일**: Canva
- **자막**: Vrew (AI 자동)

### Instagram

#### 컨셉
- **카드뉴스**: 정보 압축
- **릴스**: 짧은 팁, 경험담

#### 피드 구성
```
[카드뉴스] [카드뉴스] [릴스]
[릴스]     [카드뉴스] [카드뉴스]
[카드뉴스] [릴스]     [카드뉴스]
```

#### 해시태그 전략
```
필수 (5개):
#골린이 #골프초보 #골프시작 #골프입문 #Golfearn

추가 (5개):
#스크린골프 #골프연습 #골프레슨 #골프장비 #42살골린이
```

#### 스토리 활용
- 매일 1-2개 스토리
- 투표, Q&A 활용
- 하이라이트 정리

### X (Twitter)

#### 컨셉
- **스레드**: 심층 정보 (주 1회)
- **단일 트윗**: 짧은 팁, 공감 (매일)

#### 트윗 유형
1. **정보형**: "골린이 연습장 팁 👇"
2. **공감형**: "연습장에서 창피했던 순간..."
3. **질문형**: "같은 분 계시나요?"
4. **CTA형**: "같이 칠 골린이 찾기 → golfearn.com"

#### 성장 전략
- 골프 관련 계정 팔로우/RT
- 해시태그 검색 후 답글
- 트렌드 활용

### 블로그

#### 컨셉
- **네이버**: 검색 유입 (SEO)
- **티스토리**: 애드센스 수익

#### SEO 키워드
```
메인: 골린이, 골프 시작, 골프 입문, 골프 초보
롱테일: 골프 시작 비용, 골린이 클럽 추천, 혼자 조인, 중고 골프채
```

#### 글 구조
1. 제목 (키워드 포함)
2. 목차
3. 본문 (2,000-4,000자)
4. 이미지 (최소 3장)
5. CTA (Golfearn 링크)
6. 해시태그

### 네이버 카페

#### 전략
- **직접 홍보 X** → 자연스러운 정보 공유
- 10글 중 1-2글만 Golfearn 언급
- 댓글로 활발히 소통

#### 활동 루틴
1. 매일 10분: 새 글 읽고 댓글
2. 주 1회: 정보글 작성
3. 월 1회: 경험담 + Golfearn 소개
