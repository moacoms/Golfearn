## Phase 2: n8n 자동화 워크플로우

### 워크플로우 1: 주간 콘텐츠 생성
```
트리거: 매주 월요일 오전 9시
    │
    ▼
[Claude API 호출]
- MARKETING_PROMPT.md 로드
- 주간 주제 선정 (골린이 키워드 기반)
    │
    ▼
[브리프 생성]
- golfearn-planner 실행
- marketing-outputs/brief-*.md 저장
    │
    ▼
[콘텐츠 생성 - 병렬]
├── youtube-writer → YouTube 대본
├── instagram-writer → 카드뉴스
├── blog-writer → 블로그 글
├── thread-writer → X 스레드
└── cafe-writer → 카페 글
    │
    ▼
[품질 검수]
- golfearn-reviewer 실행
- 승인/수정 요청
    │
    ▼
[Notion/Airtable 저장]
- 콘텐츠 DB에 저장
- 발행 스케줄 설정
```

### 워크플로우 2: 자동 발행
```
트리거: 스케줄 시간 도달
    │
    ▼
[플랫폼별 발행]
├── 블로그 → 네이버 API / 티스토리 API
├── X → Twitter API
├── Instagram → Meta Graph API (예약)
└── YouTube → 수동 (대본만 제공)
    │
    ▼
[발행 기록]
- Notion/Airtable 업데이트
- Slack/카카오톡 알림
```

### 워크플로우 3: 트렌드 모니터링
```
트리거: 매일 오전 8시
    │
    ▼
[데이터 수집]
├── YouTube 골프 인기 영상
├── 네이버 골프 검색어 트렌드
└── X 골프 관련 화제
    │
    ▼
[AI 분석]
- 골린이 관점에서 콘텐츠화 가능한 주제
- 트렌드 리포트 생성
    │
    ▼
[알림]
- Slack/카카오톡으로 주간 트렌드 요약
```

## API 연동 정보

### Claude API
```
Endpoint: https://api.anthropic.com/v1/messages
Model: claude-sonnet-4-20250514
```

### YouTube Data API
- 영상 업로드: 수동 권장 (API 제한)
- 댓글 모니터링: API 가능

### Instagram Graph API
- 비즈니스 계정 필수
- 이미지/릴스 예약 발행 가능
- 승인 절차 필요 (1-2주)

### Twitter API v2
- Basic: 무료 (1,500 트윗/월)
- 스레드 자동 발행 가능

### 네이버 블로그 API
- 공식 API 없음
- Selenium/Playwright 자동화 또는 수동

### 티스토리 API
- OAuth 인증 후 자동 발행 가능
